# Res Changer

A Windows system tray utility to quickly toggle between two screen resolutions.

## Features

- Switch between two preset resolutions from the tray menu or with a configurable hotkey
- Runs in the system tray with no open windows
- Option to launch on Windows startup
- Customizable hotkey (default: `Ctrl+Alt+P`)
- Full-screen resolution picker for each preset

## Usage

1. Run `ResToggle.exe` – it appears in the system tray.
2. Right-click the icon to:
   - **Toggle Resolution** – switch between Resolution A and B
   - **Change hotkey** – set your own keyboard shortcut
   - **Resolution 1 / Resolution 2** – pick a new resolution for each preset
   - **Run on startup** – auto-launch with Windows
